JavaScript

讓網頁活起來 · 從零開始一次學會

⚡ 1. JavaScript 是什麼？

JavaScript 是網頁的「互動大腦」。HTML 是骨架，CSS 是化妝，JS 則是讓網頁可以回應點擊、計算數字、抓取資料、改變內容……沒有 JS 的網頁就像一本靜態的書。

📌 為什麼要學？因為它是全世界最流行的程式語言，不論前端還是後端都能用。

// 這是一個最簡單的 JS 程式
let message = "你好，世界！";
console.log(message); // 在開發者工具中印出訊息

📥 2. 三種引入 JavaScript 的方式

  • 行內事件：寫在 HTML 標籤的 onclick 等屬性（不推薦）
  • 內部腳本：寫在 <script> 標籤內，通常放在 <body> 底部
  • 外部腳本：獨立 .js 檔案，透過 <script src=""> 引入（最佳實務）
<!-- 外部引入範例 (寫在 HTML 底部) -->
<script src="script.js"></script>

📦 3. 變數與資料型別

變數就像「箱子」，用來存放資料。用 let 或 const 宣告。

let age = 16; // 數字
const name = "Ting"; // 文字（字串）
let isStudent = true; // 布林值（true/false）
let hobbies = ["籃球", "程式"]; // 陣列
let person = { name: "Ting", age: 16 }; // 物件

🔍 注意：const 宣告的變數不能重新賦值，let 可以改變。

🧩 4. 函式 – 把程式包起來，隨時呼叫

函式就像「食譜」，定義好步驟之後，想用隨時叫它執行。

// 定義一個打招呼的函式
function sayHello(name) {
    console.log("嗨，" + name);
}
sayHello("Ting"); // 呼叫函式，印出「嗨，Ting」

📌 箭頭函式（ES6 新語法）：const add = (a, b) => a + b;

🖱️ 5. DOM 操作 – 用 JavaScript 控制網頁元素

DOM (Document Object Model) 是 HTML 的樹狀結構，JS 可以選取元素、修改文字、樣式、甚至增刪標籤。

// 選取一個元素並改變內容
let title = document.querySelector("h1");
title.innerText = "新標題";
title.style.color = "gold"; // 改變顏色

常用方法：getElementById、querySelector、innerHTML、classList.add/remove。

🎯 6. 事件監聽 – 點擊、輸入、滑鼠移動……

事件是 JS 互動的核心。例如：按鈕被點擊時，執行某段程式。

// 選取按鈕，加上點擊事件
let btn = document.querySelector("button");
btn.addEventListener("click", () => {
    alert("你點擊了按鈕！");
});

📌 常見事件：click（點擊）、mouseover（滑入）、input（輸入內容）、submit（表單送出）。

🗂️ 7. 陣列與物件 – 管理資料的好幫手

陣列用來存放「一組清單」，物件用來描述「一個東西的詳細資訊」。

// 陣列：好朋友的名字清單
let friends = ["小明", "小華", "小美"];
console.log(friends[0]); // 印出「小明」

// 物件：一本書的資訊
let book = { title: "JS 入門", pages: 300, author: "Ting" };
console.log(book.title); // 印出「JS 入門」

常用陣列方法：push、pop、map、filter、forEach。

🔄 8. 條件判斷與迴圈 – 讓程式做決定、重複執行

if-else 根據條件執行不同程式；for/while 重複做同一件事。

// 條件判斷
let score = 85;
if (score >= 60) {
    console.log("及格！");
} else {
    console.log("加油！");
}

// for 迴圈：印出 1 到 5
for (let i = 1; i <= 5; i++) {
    console.log(i);
}

⏳ 9. 非同步 – 等待資料載入、不卡畫面

當你需要向伺服器要資料（例如天氣 API），JS 不會停下來等，而是用非同步方式處理。

// 模擬兩秒後顯示訊息 (setTimeout)
setTimeout(() => {
    console.log("2 秒後執行");
}, 2000);

// Promise 範例
let fetchData = () => {
    return new Promise((resolve) => {
        setTimeout(() => resolve("資料載入完成"), 1000);
    });
};
fetchData().then(console.log);

🔥 現代寫法更推薦 async/await，讓非同步程式像同步一樣好讀。

✅ 10. 你學會了嗎？動手勾選＋實作

  • 我知道 JavaScript 是用來讓網頁產生互動的。
  • 我會宣告變數（let / const）並區別數字、字串、布林值。
  • 我會寫函式（function）並呼叫它。
  • 我會用 querySelector 選取網頁元素，並修改內容或樣式。
  • 我會用 addEventListener 監聽按鈕點擊事件。
  • 我理解陣列與物件的簡單用法。
  • 我會寫 if-else 條件判斷和 for 迴圈。
  • 我知道非同步概念（setTimeout、Promise 的初步印象）。

🎯 實作小挑戰：做一個按鈕，點擊後讓畫面上的文字變成「你點到我了！」。

💡 提示：先用 HTML 做一個 <button> 和一個 <p>，再用 JS 監聽 click 事件，改變該段落的 innerText。
