CSS3

讓網頁變漂亮 · 從零開始一次學會

🎨 1. CSS 是什麼？

CSS (Cascading Style Sheets) 是網頁的「化妝師」。HTML 負責內容結構，CSS 負責顏色、字體、位置、動畫……讓網頁從黑白文字變成精美設計。

📌 一句話：HTML 是房子的鋼筋水泥，CSS 是油漆、窗簾和家具擺設。

h1 {
  color: gold;
  font-size: 32px;
}

↑ 這段 CSS 會讓所有 <h1> 標題變成金色、字體 32px。

📥 2. 三種引入 CSS 的方式

  • 行內樣式：直接寫在 HTML 標籤的 style 屬性（不推薦，難維護）
  • 內部樣式表：寫在 <style> 標籤內（適合單頁教學）
  • 外部樣式表：獨立 .css 檔案，透過 <link> 引入（最專業、最常用）
<!-- 外部樣式表範例 (寫在 HTML 的 head 中) -->
<link rel="stylesheet" href="style.css">

🎯 3. CSS 選擇器 — 你要修改哪個元素？

選擇器範例說明
元素選擇器p { }選取所有 <p> 段落
類別選擇器.card { }選取所有 class="card" 的元素
ID 選擇器#header { }選取 id="header" 的唯一元素
後代選擇器div p { }選取 <div> 內的所有 <p>
偽類a:hover { }滑鼠移到連結上的樣式
.btn:hover {
  background-color: #D4AF37;
  transform: scale(1.05);
}

📦 4. 盒模型 – 每一個元素都是一個盒子

在 CSS 中，每個 HTML 元素都被視為一個矩形盒子，包含：內容 (content) → 內距 (padding) → 邊框 (border) → 外邊距 (margin)。

div {
  width: 200px;
  padding: 20px;
  border: 2px solid gold;
  margin: 10px;
}

⭐ 祕訣：加上 box-sizing: border-box; 可以讓寬度包含 padding 和 border，不容易跑版。

🧩 5. Flexbox – 輕鬆搞定水平／垂直對齊

彈性盒子可以讓元素自動排列、置中、均分空間。只需要在父層設定 display: flex;。

.container {
  display: flex;
  justify-content: center; /* 水平置中 */
  align-items: center; /* 垂直置中 */
  gap: 20px; /* 子元素間距 */
}

📌 常用屬性：flex-direction（主軸方向）、flex-wrap（換行）、justify-content（主軸對齊）。

🗂️ 6. CSS Grid – 網格佈局

Grid 適合製作整個頁面的區塊劃分（例如導覽、側邊欄、主內容）。

.grid-container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  gap: 16px;
}

💡 fr 是剩餘空間的比例單位，非常實用。

✨ 7. 過渡與動畫 – 讓網頁活起來

transition 讓屬性變化有漸變效果；@keyframes 可製作完整動畫。

button {
  transition: all 0.3s ease;
}
button:hover {
  transform: translateY(-3px);
  box-shadow: 0 8px 20px gold;
}

@keyframes fadeIn {
  0% { opacity: 0; }
  100% { opacity: 1; }
}

📱 8. 響應式設計 – 手機、平板、電腦自動調整

使用 @media 查詢，根據螢幕寬度切換樣式。

@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}

📌 常用斷點：手機 ≤ 768px，平板 768px ~ 1024px，桌機 ≥ 1024px。

✅ 9. 你學會了嗎？動手勾選＋實作

  • 我知道 CSS 是用來設計網頁外觀的。
  • 我會使用類別選擇器（.class）和 ID 選擇器（#id）。
  • 我理解盒模型（margin, border, padding, content）。
  • 我會用 Flexbox 做出水平置中、垂直置中。
  • 我知道 Grid 的基本用法。
  • 我會寫簡單的 transition 或 keyframes 動畫。
  • 我知道如何用 @media 讓網頁在手機上變好閱讀。

🎯 實作小挑戰：試著用 Flexbox 把三個區塊水平排列，並在手機上改成垂直排列。
