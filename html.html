HTML5

從零開始 · 每個人都能輕鬆做出網頁

📄 1. 什麼是 HTML？

HTML 全名「超文本標記語言」，它是網頁的「骨架」。就像蓋房子需要鋼筋水泥，HTML 負責把文字、圖片、按鈕全部放在正確的位置。

🎯 一個簡單的比喻：網頁是一本圖畫書，HTML 是紙張和頁碼，CSS 是著色與排版，JavaScript 是翻頁後的音效與互動。

👉 學習 HTML 其實就是在學「<標籤>」，瀏覽器看到標籤就知道該怎麼顯示內容。

🏗️ 2. 每個網頁都有的「骨架」

一個標準的 HTML 文件長這樣，就像人體有頭、身體、腳：

<!DOCTYPE html> ← 宣告這是 HTML5
<html> ← 整份網頁的根
  <head> ← 頭部（放標題、樣式設定）
    <title>我的第一個網頁</title>
  </head>
  <body> ← 身體（所有看得到的內容）
    <h1>哈囉！世界！</h1>
  </body>
</html>

✏️ 只要把這段複製到記事本，存成 index.html，用瀏覽器打開就會看到「哈囉！世界！」的大標題。

🏷️ 3. 常用標籤大全（像樂高積木）

學會這些標籤，你已經可以做出 80% 的靜態網頁：

標籤用途範例
<h1> ~ <h6>標題（由大到小）<h1>大標題</h1>
<p>段落文字<p>這是一段文字...</p>
<a>超連結<a href="https://google.com">點我去Google</a>
<img>插入圖片<img src="photo.jpg" alt="描述">
<div>區塊容器（用來分組）<div class="box">內容</div>
<span>行內容器<span style="color:gold">金色字</span>
<!-- 這是註解，不會顯示在網頁上，只給寫程式的人看 -->

🧩 4. HTML5 新語意標籤（讓網頁結構一目了然）

過去大家習慣用一堆 <div>，HTML5 新增了更直覺的標籤：

  • <header> — 頁首（通常放標題、導覽）
  • <nav> — 導覽列選單
  • <main> — 主要內容區域（整頁最重要）
  • <section> — 章節區塊
  • <article> — 獨立的文章／內容
  • <aside> — 側邊欄
  • <footer> — 頁尾（版權、聯絡方式）
<body>
  <header>我的網站標題</header>
  <nav><a>首頁</a> <a>關於</a></nav>
  <main><article>文章內容...</article></main>
  <footer>© 2025 Ting</footer>
</body>

🔍 搜尋引擎會優先看懂這些語意標籤，對 SEO 很有幫助。

📝 5. 表單 — 讓網頁可以問問題、收集資料

你想做「聯絡我們」或「登入頁面」嗎？就用表單：

<form action="/submit" method="post">
  <label>姓名：<input type="text" name="name"></label>
  <label>電子郵件：<input type="email" name="email"></label>
  <button type="submit">送出</button>
</form>

✅ 常用輸入類型：text（文字）、email（電子信箱）、number（數字）、radio（單選）、checkbox（複選）。

✅ 6. 你學會了嗎？動手勾選＋試試看

  • 我知道 HTML 是網頁的骨架，是用標籤組成的。
  • 我能寫出 <!DOCTYPE html>、<html>、<head>、<body> 的基本結構。
  • 我會用 <h1>、<p>、<a>、<img>、<div> 做出簡單網頁。
  • 我知道 <header>、<nav>、<main>、<footer> 的用途。
  • 我會做一個簡單的表單（輸入框 + 按鈕）。
  • 我能自己寫出一個自我介紹的網頁（至少包含標題、一段自我介紹、一張圖片、一個超連結）。

🎉 勾選完前三項，你已經贏過 80% 的初學者！全部勾選後，你已經具備獨立製作靜態網站的能力。

💡 小挑戰：試著修改上面的範例，把「我的第一個網頁」改成你自己的名字，看看瀏覽器標題會不會改變。
